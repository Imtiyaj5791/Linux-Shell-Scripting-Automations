# Day 05: Loops and Arrays in Bash

## 1. For Loop
Used when you know the exact number of times you want to iterate or loop through a specific list of items. It executes the block of code for a fixed set of elements.

## 2. While Loop
Used to repeat a block of code as long as a specified condition remains true. It is commonly used when the exact number of iterations is unknown beforehand or for reading files line-by-line.

## 3. Arrays
An array is a data structure used to store multiple values or a list of items inside a single variable, which can be accessed using index numbers starting from 0. It helps in clean automation by avoiding the creation of multiple individual variables.
